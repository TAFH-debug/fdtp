+------------------------------------------------------------------+
|               Request structure                                   |
+--------------+-----------------+---------------+-----------------+
| 1 byte       | 2-5 bytes       | 6-9 bytes     |                 |
+--------------+-----------------+---------------+-----------------+
| Method type  | Method metadata | URI offset    |                 |
|              | offset          |               |                 |
+--------------+-----------------+---------------+-----------------+

Fixed size requests header.
+------------------------------------------------+
|               Request structure                |
|                (SHORT)                         |
+--------------+-----------+---------------------+
| 1 byte       | 2-9 bytes |   10-25 bytes       |
+--------------+-----------+---------------------+
| Method type  | URID      |   Data              |
|              |           |                     |
+--------------+-----------+---------------------+
| Fixed size: 1 + 8 + 16 = 25 bytes              |
+------------------------------------------------+

+------------------------------------------------+
|               Request structure                |
|                (LONG)                          |
+--------------+-----------+---------------------+
| 1 byte       | 2-9 bytes |   10-41 bytes       |
+--------------+-----------+---------------------+
| Method type  | URID      |   Data              |
|              |           |                     |
+--------------+-----------+---------------------+
| Fixed size: 1 + 8 + 32 = 41 bytes              |
+------------------------------------------------+

Request type code(RTC):
GET - 0
SHORT - 1
LONG - 2

URID = Unique Request IDentificator.
